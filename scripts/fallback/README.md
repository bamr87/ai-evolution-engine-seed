# Fallback Scripts

Emergency fallback and recovery scripts for system resilience and error handling.

## Scripts Overview

| Script | Purpose |
|--------|---------|
| `emergency-fallback.sh` | Emergency fallback for critical failures |
| `simple-ai-simulator.sh` | Simple AI growth simulator |
| `simple-change-applier.sh` | Simple change application |
| `simple-context-collector.sh` | Simple context collection |

## Features

- Emergency recovery mechanisms
- Simplified fallback operations
- System resilience support
- Critical failure handling

## Usage

```bash
# Activate emergency fallback
./emergency-fallback.sh --mode recovery

# Use simple simulator
./simple-ai-simulator.sh --prompt "Basic improvement"
```
